Concept of Vectors
Understanding what vectors are and why they matter
1.1 What Is a Vector?

In everyday life, we describe many quantities simply by their size — temperature, mass, time, speed. These are called scalar quantities. But many physical phenomena cannot be fully described by magnitude alone. When we say "a car is moving at 60 km/h," we might also need to know in which direction it is moving. This is where vectors become essential.

Definition — Vector

A vector is a mathematical object that has both magnitude (size) and direction. It is represented geometrically as a directed line segment (an arrow), where the length of the arrow represents the magnitude and the arrowhead indicates the direction.

Definition — Scalar

A scalar is a quantity described by magnitude only, with no associated direction. Examples include temperature (25°C), mass (10 kg), and time (5 seconds).

1.2 Notation and Representation

Vectors are written using bold letters such as a, b, v, or with an arrow overhead such as →a. The magnitude of vector a is written as |a| or simply a (italicised).

A vector from point A to point B is written as →AB. Point A is the initial point (tail) and point B is the terminal point (head).

1.4 Physical Examples of Vectors

Vectors appear constantly in physics and engineering. Key examples include:

Displacement — A person walks 5 km North. Both the 5 km and the direction "North" are needed. Compare this to distance (scalar), which only says "5 km."

Velocity — A ball moves at 10 m/s to the East. Speed (10 m/s) is scalar; velocity (10 m/s East) is a vector.

Force — Gravity pulls you downward with a force of 600 N. The downward direction is critical.

Acceleration, momentum, electric field, and magnetic flux are all vectors.

1.5 Column Vector Notation

In 2D space, a vector can be written as a column matrix. If a vector v moves 3 units in the x-direction and 4 units in the y-direction:

v = [ 3 ] ← x-component [ 4 ] ← y-component General 2D form: v = (x, y) or v = xî + yĵ General 3D form: v = (x, y, z) or v = xî + yĵ + zk̂
1.6 Geometric Interpretation

Geometrically, a vector is drawn as an arrow. The key insight is that a vector does not have a fixed location — only its magnitude and direction matter. Two arrows of equal length pointing in the same direction represent the same vector, regardless of where they are drawn on a page. This property is called free vector.

Worked Example 1.1

Question: A displacement vector goes from point A(1, 2) to point B(4, 6). Find the vector →AB and its magnitude.

1
Find vector components: Subtract initial from terminal point.
→AB = B − A = (4−1, 6−2) = (3, 4)
2
Find magnitude: Use Pythagorean theorem.
|→AB| = √(3² + 4²) = √(9 + 16) = √25 = 5
3
Conclusion: The vector is (3, 4) with magnitude 5 units.
Key Insight: Equal vectors have the same direction and magnitude but can be positioned anywhere in space. The position does not define the vector — only its components do.

Representation of Vectors in 3 Directions
Mastering 3-dimensional vector space with i, j, k components
2.1 Three-Dimensional Space

While 2D vectors are useful, the real world exists in three dimensions. To represent any point or direction in 3D space, we need three coordinate axes: the x-axis (horizontal), y-axis (vertical), and z-axis (depth). These three axes are mutually perpendicular — each is at 90° to the others.

Definition — 3D Vector

A 3D vector is written as v = (x, y, z) where x, y, and z are the components along the three coordinate axes. Equivalently, v = xî + yĵ + zk̂, where î, ĵ, k̂ are unit vectors along x, y, z axes respectively.

2.2 The Standard Basis Vectors

The three standard basis vectors are the building blocks of 3D vector space. Every vector in 3D can be written as a combination of these:

î = (1, 0, 0) ← unit vector along x-axis ĵ = (0, 1, 0) ← unit vector along y-axis k̂ = (0, 0, 1) ← unit vector along z-axis Any vector: v = xî + yĵ + zk̂ = (x, y, z) Example: (3, -2, 5) = 3î − 2ĵ + 5k̂
2.3 Magnitude in 3D

The magnitude (length) of a 3D vector extends the Pythagorean theorem into the third dimension. For a vector v = (x, y, z):

|v| = √(x² + y² + z²) Example: v = (2, 3, 6) |v| = √(2² + 3² + 6²) = √(4 + 9 + 36) = √49 = 7
2.4 Direction Cosines

A 3D vector makes angles with each of the three axes. These angles α, β, γ (alpha, beta, gamma) and their cosines are called direction cosines. They describe the orientation of the vector in space.

cos α = x/|v| ← angle with x-axis cos β = y/|v| ← angle with y-axis cos γ = z/|v| ← angle with z-axis Key property: cos²α + cos²β + cos²γ = 1 (always true!)
Worked Example 2.1 — Direction Cosines

Find the direction cosines of v = (2, 3, 6)

1
Find magnitude: |v| = √(4+9+36) = √49 = 7
2
cos α = 2/7, cos β = 3/7, cos γ = 6/7
3
Verify: (2/7)² + (3/7)² + (6/7)² = 4/49 + 9/49 + 36/49 = 49/49 = 1 ✓
2.5 Position Vectors

Every point P(x, y, z) in 3D space has a position vector →OP from the origin O(0,0,0) to P. This gives a unique vector for every location in space.

P = (a, b, c) → position vector →OP = aî + bĵ + ck̂ Vector between two points: →AB = B − A = (b₁−a₁)î + (b₂−a₂)ĵ + (b₃−a₃)k̂
Worked Example 2.2 — Vector Between Points

A = (1, 2, 3) and B = (4, 0, 7). Find vector AB and its magnitude.

1
→AB = (4−1)î + (0−2)ĵ + (7−3)k̂ = 3î − 2ĵ + 4k̂
2
|→AB| = √(3² + (−2)² + 4²) = √(9 + 4 + 16) = √29 ≈ 5.39
2.6 Adding Vectors in 3D

3D vector addition works component by component, exactly like 2D but with an extra z-component:

a + b = (a₁+b₁, a₂+b₂, a₃+b₃) Example: (1,2,3) + (4,5,6) = (5,7,9)
Right-Hand Rule: To set up a 3D coordinate system correctly, point your right hand's fingers along x, curl them toward y — your thumb points in the z direction. This is the standard right-handed coordinate system used in mathematics and physics.

3.1 The Dot Product

One of the most powerful tools in vector mathematics is the dot product (also called the scalar product or inner product). It multiplies two vectors and produces a single number (scalar). Its geometric meaning is deeply connected to the angle between the vectors.

Algebraic definition: a · b = a₁b₁ + a₂b₂ + a₃b₃ Geometric definition: a · b = |a| |b| cos θ Therefore, angle formula: cos θ = (a · b) / (|a| |b|) θ = arccos[ (a · b) / (|a| |b|) ]
3.2 Properties of the Dot Product

3.3 Basis Vector Dot Products
î·î = 1, ĵ·ĵ = 1, k̂·k̂ = 1 (parallel, angle=0°) î·ĵ = 0, ĵ·k̂ = 0, î·k̂ = 0 (perpendicular, angle=90°)
3.4 Finding the Angle — Full Process
Worked Example 3.1

Find the angle between a = (1, 2, 2) and b = (2, 1, -2)

1
Compute dot product: a·b = (1)(2)+(2)(1)+(2)(−2) = 2+2−4 = 0
2
Find magnitudes: |a| = √(1+4+4) = 3 and |b| = √(4+1+4) = 3
3
Apply formula: cos θ = 0/(3·3) = 0
4
Result: θ = arccos(0) = 90°. These vectors are perpendicular!
3.5 Projection of a Vector

The scalar projection of vector a onto vector b is how much of a lies in the direction of b. The vector projection gives the actual projected vector.

Scalar projection of a onto b: comp_b(a) = (a · b) / |b| Vector projection of a onto b: proj_b(a) = [(a · b) / |b|²] · b
Worked Example 3.2 — Projection

Find the projection of a = (3, 4) onto b = (1, 0)

1
a·b = 3(1)+4(0) = 3
2
|b|² = 1² + 0² = 1
3
proj_b(a) = (3/1)(1,0) = (3,0)
4
Geometric meaning: the shadow of (3,4) on the x-axis is (3,0). Makes sense!
Orthogonality Test: If a·b = 0, the vectors are perpendicular (orthogonal). This is a fast and powerful test used constantly in linear algebra and physics.

Vector Operations
Sum, subtraction, scalar multiplication, and the cross product
4.1 Vector Addition

Vectors are added component by component. Geometrically, this corresponds to the triangle law or parallelogram law of addition.

a + b = (a₁+b₁, a₂+b₂, a₃+b₃) Parallelogram Law: Place vectors tail-to-tail. The diagonal of the parallelogram = a + b Triangle Law: Place head-to-tail. The closing side from first tail to last head = a + b
4.2 Vector Subtraction

Subtracting vector b from a is the same as adding the negative of b:

a − b = a + (−b) = (a₁−b₁, a₂−b₂, a₃−b₃) −b has the same magnitude as b but opposite direction
4.3 Scalar Multiplication

Multiplying a vector by a scalar (number) scales its magnitude. If the scalar is negative, it also reverses the direction.

λa = (λa₁, λa₂, λa₃) |λa| = |λ| · |a| If λ > 0: same direction, scaled magnitude If λ < 0: opposite direction, scaled magnitude If λ = 0: zero vector If λ = 1: unchanged If λ = -1: reversed direction (negative vector)
4.4 The Cross Product

Unlike the dot product (which gives a scalar), the cross product of two 3D vectors gives a new vector that is perpendicular to both original vectors.

a × b = | î ĵ k̂ | | a₁ a₂ a₃ | | b₁ b₂ b₃ | = î(a₂b₃ − a₃b₂) − ĵ(a₁b₃ − a₃b₁) + k̂(a₁b₂ − a₂b₁) |a × b| = |a| |b| sin θ (area of parallelogram)
Worked Example 4.1 — Cross Product

Find a × b where a = (1, 2, 3) and b = (4, 5, 6)

1
î component: (2)(6) − (3)(5) = 12 − 15 = −3
2
ĵ component: −[(1)(6) − (3)(4)] = −[6−12] = −(−6) = 6
3
k̂ component: (1)(5) − (2)(4) = 5 − 8 = −3
4
Result: a × b = (−3, 6, −3)
5
Verify perpendicular: (−3,6,−3)·(1,2,3) = −3+12−9 = 0 ✓

4.6 Linear Combinations

A linear combination of vectors uses both addition and scalar multiplication to create new vectors:

r = αa + βb + γc (α,β,γ are scalars) Example: r = 2(1,0,1) + 3(0,1,0) − (1,1,1) = (2,0,2) + (0,3,0) − (1,1,1) = (2+0−1, 0+3−1, 2+0−1) = (1, 2, 1)
Right-Hand Rule for Cross Product: Point your right hand's fingers from a toward b, curl them. Your thumb points in the direction of a×b.

Unit Vectors & Magnitude
Normalisation, direction, and vector decomposition
5.1 What Is a Unit Vector?

A unit vector is a vector with a magnitude of exactly 1. Unit vectors capture pure direction — they tell you where something points without saying how far. They are denoted with a hat: â (read "a-hat").

Unit vector in direction of a: â = a / |a| Proof it has magnitude 1: |â| = |a/|a|| = |a|/|a| = 1 ✓
5.2 Normalising a Vector

The process of converting a vector to a unit vector is called normalisation or normalizing. It is one of the most common operations in vector mathematics, physics, and computer graphics.

Worked Example 5.1 — Normalisation

Find the unit vector in the direction of v = (3, 4, 0)

1
Find magnitude: |v| = √(9 + 16 + 0) = √25 = 5
2
Divide each component by magnitude: v̂ = (3/5, 4/5, 0/5) = (0.6, 0.8, 0)
3
Verify: |v̂| = √(0.36 + 0.64 + 0) = √1 = 1 ✓
5.3 Magnitude in 2D and 3D
2D magnitude: |v| = √(x² + y²) 3D magnitude: |v| = √(x² + y² + z²) nD magnitude: |v| = √(v₁² + v₂² + ... + vₙ²) Also called: length, norm, or Euclidean distance from origin
5.4 Vector Decomposition

Any vector can be decomposed (broken down) into components along the coordinate axes. This is the reverse of combining unit vectors:

v = vₓî + vyĵ + vzk̂ where: vₓ = |v|cos α (x-component) vy = |v|cos β (y-component) vz = |v|cos γ (z-component) In 2D: vₓ = |v|cosθ, vy = |v|sinθ
5.5 Application: Finding Vector from Magnitude and Direction
Worked Example 5.2

A force has magnitude 10 N at angle 30° to the x-axis in the xy-plane. Write as a vector.

1
x-component: Fx = 10 cos(30°) = 10 × (√3/2) = 5√3 ≈ 8.66 N
2
y-component: Fy = 10 sin(30°) = 10 × (1/2) = 5 N
3
F = (5√3)î + 5ĵ ≈ 8.66î + 5ĵ
4
Verify: |F| = √((5√3)²+5²) = √(75+25) = √100 = 10 ✓

Application of Vectors in Differentiation
Vector calculus, derivatives, curves, and rates of change
6.1 Vector-Valued Functions

A vector-valued function assigns a vector to each value of a parameter (usually time t). Instead of a single output, it gives a vector output. These functions describe motion, curves, and changing directions in space.

r(t) = f(t)î + g(t)ĵ + h(t)k̂ Example: position of a particle: r(t) = t²î + sin(t)ĵ + e^t k̂ At t=0: r(0) = 0î + 0ĵ + 1k̂ = (0, 0, 1)
6.2 Differentiating a Vector Function

The derivative of a vector function is taken component by component, just like differentiating each scalar component independently. This gives the rate of change of the vector.

If r(t) = f(t)î + g(t)ĵ + h(t)k̂ Then: r'(t) = f'(t)î + g'(t)ĵ + h'(t)k̂ This is the velocity vector if r(t) is position.
Worked Example 6.1 — Differentiation

Given r(t) = (t³)î + (4t²)ĵ + (2t)k̂, find velocity r'(t) and acceleration r''(t).

1
Velocity: r'(t) = 3t²î + 8tĵ + 2k̂
2
Acceleration: r''(t) = 6tî + 8ĵ + 0k̂ = 6tî + 8ĵ
3
At t=1: r'(1) = 3î + 8ĵ + 2k̂, speed = √(9+64+4) = √77
6.3 Tangent Vectors

The derivative r'(t) gives the tangent vector to the curve at each point. It points in the direction the curve is moving at that instant. The unit tangent vector T(t) = r'(t)/|r'(t)|.

6.4 Rules for Vector Derivatives
Sum rule: d/dt [a(t) + b(t)] = a'(t) + b'(t) Scalar multiplication: d/dt [f(t)a(t)] = f'(t)a(t) + f(t)a'(t) Dot product: d/dt [a·b] = a'·b + a·b' Cross product: d/dt [a×b] = a'×b + a×b' (order matters!)
6.5 Arc Length of a Vector Curve

The arc length (total distance along a curve) from t=a to t=b is found by integrating the speed:

L = ∫ₐᵇ |r'(t)| dt = ∫ₐᵇ √(f'² + g'² + h'²) dt

Lines, planes, distances, and geometric applications of vectors
7.1 Equation of a Line in 3D

In 3D, a line is uniquely determined by a point on the line and a direction vector. There are three standard forms for a line equation.

Vector form: r = a + td (t is any real number) Parametric form: x = a₁ + td₁ y = a₂ + td₂ z = a₃ + td₃ Symmetric form (if d₁,d₂,d₃ ≠ 0): (x−a₁)/d₁ = (y−a₂)/d₂ = (z−a₃)/d₃ = t where a = (a₁,a₂,a₃) is a point, d = (d₁,d₂,d₃) is direction
7.2 Equation of a Plane

A plane in 3D is uniquely determined by a point on the plane and a normal vector (perpendicular to the plane).

If n = (A, B, C) is normal and P₀(x₀,y₀,z₀) is on the plane: A(x−x₀) + B(y−y₀) + C(z−z₀) = 0 Standard form: Ax + By + Cz = D where D = Ax₀ + By₀ + Cz₀ Vector dot product form: n · (r − a) = 0
Worked Example 7.1 — Plane Equation

3

7.3 Distance from a Point to a Plane
Distance from point P(x₀,y₀,z₀) to plane Ax+By+Cz=D: d = |Ax₀ + By₀ + Cz₀ − D| / √(A² + B² + C²)
7.4 Distance Between Two Points
d(A, B) = |→AB| = √((b₁−a₁)² + (b₂−a₂)² + (b₃−a₃)²) Example: A(1,2,3), B(4,6,3) d = √((4−1)² + (6−2)² + (3−3)²) = √(9+16+0) = √25 = 5
7.5 Midpoint Formula
Midpoint M of segment AB: M = (A + B) / 2 = ((a₁+b₁)/2, (a₂+b₂)/2, (a₃+b₃)/2)
7.6 Angle Between Two Planes

The angle between two planes equals the angle between their normal vectors:

cos θ = |n₁ · n₂| / (|n₁| |n₂|) Planes are parallel if n₁ × n₂ = 0 (normals parallel) Planes are perpendicular if n₁ · n₂ = 0 (normals perpendicular)

The point dividing segment AB in ratio m:n from A:
